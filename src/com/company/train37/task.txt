Задание All Star Code # 18

Создайте функцию с именем, которая принимает 2 строковых аргумента и возвращает
целое число из количества вхождений, в котором второй аргумент находится в первом.

Если вхождения не могут быть найдены, должно быть возвращено значение 0.

strCount("Hello", 'o') // => 1
strCount("Hello", 'l') // => 2
strCount("", 'z')      // => 0
Заметки:

Первый аргумент может быть пустой строкой
Второй строковый аргумент всегда будет иметь длину 1
